# 📝 Todo App

A simple and clean Todo application built with HTML, CSS, and JavaScript.

## 🚀 Features
- Add tasks
- Mark tasks as completed
- Edit tasks
- Delete tasks
- Filter (All / Active / Completed)
- Clear completed tasks
- Persistent storage using localStorage

## 🛠️ Technologies
- HTML
- CSS
- JavaScript (Vanilla)
